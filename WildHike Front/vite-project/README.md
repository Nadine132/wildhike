# WildHike Frontend

El frontend de WildHike está desarrollado con React y Vite. Proporciona la interfaz de usuario para la plataforma y se encarga de la interacción con la API del backend.

## Tecnologías Utilizadas

- **React**: Biblioteca para construir interfaces de usuario.
- **Vite**: Herramienta de construcción rápida y desarrollo para React.
- **TailwindCSS**: Framework de CSS para estilos rápidos y responsivos.
