# WildHike Backend

El backend de WildHike está construido con Node.js y Express. Proporciona la API que interactúa con la base de datos y maneja la lógica de la aplicación.

## Tecnologías Utilizadas

- **Node.js**: Entorno de ejecución para JavaScript en el servidor.
- **Express**: Framework para Node.js que facilita el desarrollo de la API.
- **PostgreSQL**: Base de datos relacional.
- **Sequelize**: ORM para interactuar con la base de datos.
- **JWT**: JSON Web Tokens para autenticación.
